# LangChain Chains

This project demonstrates a simple AI workflow using Google Gemini and LangChain.

## Features

- Explain a topic
- Summarize the explanation
- Translate the summary into Urdu

## Technologies

- Python
- LangChain
- Google Gemini
- python-dotenv

## Installation

pip install -r requirements.txt

## Run

python app.py
