# 📊 AI CSV Data Analysis Agent

An AI-powered CSV Data Analysis Assistant built using **Python**, **Pandas**, and **Google Gemini**. This application allows users to analyze CSV files by asking questions in natural language and receiving intelligent, human-readable answers.

---

## 🚀 Features

- 📂 Read CSV Files
- 🤖 AI-powered Data Analysis
- 💬 Ask Questions in Natural Language
- 📊 Automatic Data Insights
- 🧠 Google Gemini Integration
- ⚡ Fast CSV Processing with Pandas
- 🔒 Secure API Key Management
- 🧩 Simple and Modular Code

---

## 🛠️ Tech Stack

- Python
- Pandas
- Google Gemini API
- LangChain
- Python Dotenv

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/adilali48/AI-Portfolio.git
```

### 2. Navigate to the Project Folder

```bash
cd 26-AI-CSV-Agent
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## ▶️ Run the Project

```bash
python app.py
```
