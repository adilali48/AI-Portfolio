# 🤖 Gemini AI Chatbot

A chatbot built using Python, Streamlit, and Google Gemini API.

## Features

- AI-powered chatbot
- Streamlit interface
- Chat history
- Secure API key with .env

## Technologies

- Python
- Streamlit
- Google Gemini API

## Installation

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
