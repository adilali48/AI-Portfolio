# 🌐 AI Web Search Agent

An AI-powered Web Search Agent built with **LangChain**, **Google Gemini**, and **Tavily Search API**. This application searches the internet in real time, gathers relevant information, and uses Gemini to generate accurate, up-to-date answers.

---

## 🚀 Features

- 🌐 Real-time Web Search
- 🤖 Google Gemini Integration
- 🔍 Tavily Search API
- 📚 AI-powered Answer Generation
- ⚡ Fast and Lightweight
- 🛡️ Secure API Key Management
- 🧩 Modular Project Structure

---

## 🛠️ Tech Stack

- Python
- LangChain
- Google Gemini API
- Tavily Search API
- Python Dotenv

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/adilali48/AI-Portfolio.git
```

### 2. Navigate to the Project

```bash
cd 23-AI-Web-Search-Agent
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
TAVILY_API_KEY=YOUR_TAVILY_API_KEY
```

---

## ▶️ Run the Application

```bash
python app.py
```
