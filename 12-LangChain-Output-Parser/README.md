# LangChain Output Parser

A beginner-friendly AI project demonstrating how to generate structured responses using LangChain Prompt Templates and Google Gemini AI.

## Features

- Google Gemini AI
- LangChain PromptTemplate
- Structured AI Output
- Dynamic User Input

## Technologies

- Python
- LangChain
- Google Gemini
- python-dotenv

## Installation

pip install -r requirements.txt

## Run

python app.py
