# 🎤 AI Voice Assistant

An AI-powered Voice Assistant built using **Python**, **OpenAI Whisper**, **Groq Llama 3.3**, and **pyttsx3**. The assistant records your voice, converts speech to text using Whisper, generates intelligent responses using Groq, and speaks the response back.

---

## 🚀 Features

- 🎤 Voice Recording
- 📝 Speech-to-Text using Whisper
- 🤖 AI Responses using Groq Llama 3.3
- 🔊 Text-to-Speech Output
- ⚡ Fast AI Inference
- 💬 Interactive Voice Conversation
- 🔒 Secure API Key Management

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/adilali48/AI-Portfolio.git
```

### 2. Navigate to Project

```bash
cd 28-AI-Voice-Assistant
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

---

## ▶️ Run the Project

```bash
python app.py
```
