# RAG Chatbot with Gemini

A complete Retrieval-Augmented Generation (RAG) chatbot built using LangChain, ChromaDB, and Google Gemini.

## Features

- Load PDF
- Split Text
- Generate Embeddings
- Store in ChromaDB
- Semantic Search
- AI Answers with Gemini

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Tech Stack

- Python
- LangChain
- ChromaDB
- Google Gemini
- PyPDF
